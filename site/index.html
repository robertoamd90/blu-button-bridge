Supported hardware

Current target

ESP32 DevKit V1 with ESP-WROOM-32 and 4 MB flash.

ESP32 4 MB flash USB data cable

Other ESP32-family boards may connect, but this installer is currently published for the primary BluButtonBridge hardware target only.

How it works

Three-minute setup

  1. Connect the ESP32 to your computer with a USB data cable.
  2. Open this page in Chrome or Edge and click Install Latest Release.
  3. Select the serial port, confirm erase/install if prompted, and wait for flashing to finish.
  4. After reboot, join the BBB-XXXXXX access point or open the device on your local network.

After install

What to expect

  • The first-time fallback AP starts automatically when no WiFi credentials are saved.
  • The built-in web UI lets you configure WiFi, MQTT, GPIO actions, BLE devices, and OTA updates.
  • The GitHub OTA flow inside the device remains available for future upgrades.

Notes

Before you flash

  • This page requires HTTPS plus Web Serial browser support.
  • The release must include BluButtonBridge-full.bin or install will be blocked.
  • If browser install is unavailable, you can still download the same release asset and flash it manually.
